# Initiative Is The Top-Level Work

Convergence treats an **Initiative** as the top-level unit of agent-driven work, while a **Project** is the local product codebase context and a **Session** is one execution attempt. Older specs described Project as the top-level domain object because early Convergence was project-centered, but agent-native delivery often spans multiple Sessions and may span multiple Projects; Initiative-centered language better preserves that delivery story without reducing work to a repository or transcript.
