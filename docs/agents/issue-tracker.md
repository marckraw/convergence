# Issue Tracker

This repo tracks product roadmap and implementation tickets in Linear.

- Workspace project: `convergence`
- Linear project URL: `https://linear.app/marckraw/project/convergence-f66c7ae332ee`
- Default team: `marckraw` (`MAR`)

When creating issues for this repo:

- create issues in the Linear `convergence` project
- use vertical slices that are independently reviewable
- use Linear blocking relations for dependencies
- prefer milestone grouping when a feature has multiple related slices
- keep acceptance criteria in the issue body, not only in comments

GitHub Issues are not the source of truth for this repo unless a future decision changes this file.
