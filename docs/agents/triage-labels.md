# Triage Labels

This repo uses Linear as its issue tracker. The canonical triage roles are:

| Role                | Preferred Linear label |
| ------------------- | ---------------------- |
| Needs evaluation    | `needs-triage`         |
| Waiting on reporter | `needs-info`           |
| Ready for agent     | `ready-for-agent`      |
| Ready for human     | `ready-for-human`      |
| Will not fix        | `wontfix`              |

The current Linear workspace may not have all of these labels configured yet. If a label does not exist, do not invent a near-match silently; create the issue without that label and mention the missing label in the task handoff.

Existing workspace labels observed during setup:

- `independent`
- `cursor-agent`
- `education-first`
- `ai`
